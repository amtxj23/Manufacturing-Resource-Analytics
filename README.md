# рџ“– Project Overview

Modern manufacturing industries generate thousands of production records every day. Without proper analytics, organizations struggle to identify the root causes of production defects, rising operational costs, equipment downtime, and declining workforce productivity.

This project simulates the work of a Manufacturing Data Analyst by designing a relational SQL database, transforming raw manufacturing data into a structured data warehouse, and performing advanced business analytics to generate actionable insights for executive decision-making.

The complete project demonstrates how raw operational data can be converted into strategic intelligence using SQL and Business Intelligence techniques.
